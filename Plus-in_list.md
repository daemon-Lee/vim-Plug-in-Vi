# [vim-airline](https://github.com/vim-airline/vim-airline)
Plugin thay đổi giao diện cho status bar của Vim
```bash
" Cài đặt với Vim-Plug
Plug 'vim-airline/vim-airline'
Plug 'vim-airline/vim-airline-themes'
```
